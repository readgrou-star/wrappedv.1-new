# **App Name**: WrappedForm

## Core Features:

- Form Creation: Enable event organizers to create customizable registration forms with various field types and options.
- Story Design: Allow event organizers to customize the visual design of the personalized, shareable stories generated from form submissions, including templates, colors, logos, and message.  Reasoning tool to determine the applicability of each customizable visual element to user stories.
- Automated Story Generation: Automatically generate personalized stories for participants based on their form submissions.
- Social Sharing: Enable participants to easily share their personalized stories on social media platforms (Instagram, X, LinkedIn) and download the image.
- Form Analytics: Provide event organizers with analytics to track form performance, submissions, story shares, and other relevant metrics.
- Database Integration: Data storage for user-generated forms and submissions.
- Landing Page: Generate a marketing landing page for each form with examples

## Style Guidelines:

- Primary color: A vibrant indigo (#6366F1) to evoke a sense of creativity and modernity.
- Background color: Light gray (#F9FAFB) for a clean and neutral backdrop, ensuring readability and a focus on content.
- Accent color: A lively violet (#8B5CF6), used sparingly for interactive elements and highlights, complementing the indigo and adding visual interest.
- Body and headline font: 'Inter', a versatile sans-serif, provides a clean and modern look.
- Use clean and minimalist icons to represent various form elements and actions.
- A clean, minimal, mobile-first approach will be applied across all screens
- Use smooth and subtle animations (framer-motion style) for transitions and user feedback, enhancing the overall user experience.
